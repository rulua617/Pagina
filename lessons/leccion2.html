Lección 2: Etiquetas y Elementos


¿Qué son las Etiquetas en HTML?

En HTML, utilizamos etiquetas para definir elementos. Las etiquetas son como instrucciones que le dicen al navegador cómo debe mostrar el contenido en la página web. Cada etiqueta tiene un propósito específico.

Algunas Etiquetas Básicas

Veamos algunas etiquetas básicas y cómo se utilizan:

  • <h1> a <h6>: Encabezados de diferentes tamaños.
  • <p>: Párrafos para organizar el texto.
  • <a>: Enlaces para conectar páginas y recursos.
  • <img>: Imágenes para mostrar gráficos.
  • <ul> y <ol>: Listas no ordenadas y ordenadas.
  • <li>: Elementos de la lista.

Ejemplo de Código HTML

HTML 



                
                    <!DOCTYPE html>
                    <html lang="es">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Título Principal</title>
                    </head>
                    <body>
                        <h1>Título Principal</h1>
                        <p>Este es un párrafo de texto.</p>
                        <a href="https://www.ejemplo.com">Enlace de Ejemplo</a>
                        <img src="imagen.jpg" alt="Descripción de la Imagen">
                        <ul>
                            <li>Elemento 1</li>
                            <li>Elemento 2</li>
                        </ul>
                        <ol>
                            <li>Elemento A</li>
                            <li>Elemento B</li>
                        </ol>
                    </body>
                    </html>
