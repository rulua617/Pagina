Lección 1: Introducción a HTML


¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear y diseñar páginas web. Permite estructurar el contenido de una página web utilizando una serie de elementos y etiquetas. Es como el lenguaje que usamos para decirle a una computadora cómo mostrar información en una página web.

Elementos Básicos de HTML

En HTML, los elementos se crean utilizando etiquetas. Algunos elementos básicos incluyen:

  • <!DOCTYPE html>: Indica la version que usamos de HTML.
  • <html>: Define el comienzo y el final del documento HTML.
  • <head>: Contiene metadatos, como el título de la página.
  • <body>: Contiene el contenido principal de la página.
  • <title>: Establece el título de la página que aparecerá en la pestaña del navegador.
  • <h1>: Encabezado principal, en este caso, "¡Hola, Mundo!"
  • <p>: Párrafo, utilizado para el texto de la página.

Ejemplo de Código HTML

HTML 

                
                    <!DOCTYPE html>
                    <html>
                        <head>
                            <title>Mi Primera Página</title>
                        </head>
                        <body>
                            <h1>¡Hola, Mundo!</h1>
                            <p>Esta es mi primera página web.</p>
                        </body>
                    </html>


Importante

Esta fue una estructura muy basica, funciona pero aun deblemos conocer mas elementos.

Ahora veremos elementos necesarios para evitar errores.

  • <html lang="es">: Inicia el documento HTML y define el idioma (en este caso, español).
  • <head>: Contiene metadatos, como el título de la página.
  • <meta charset="UTF-8">: Define el conjunto de caracteres a UTF-8 para admitir varios idiomas y símbolos.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configura la visualización para dispositivos móviles.

Ejemplo de Código HTML

HTML 

            
                <!DOCTYPE html>
                    <html lang="es">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>¡Hola, Mundo!</title>
                        </head>
                        <body>
                            <h1>¡Hola, Mundo!</h1>
                            <p>Esta es nuestra primera página web.</p>
                        </body>
                    </html>
